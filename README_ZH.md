# IntelliAvatar  
### AI Worker OS —— 从聊天到真正的桌面自主操作

IntelliAvatar **不是一个聊天机器人**。  
它是一个 **AI 办公操作系统（AI Worker OS）**，能够在真实电脑上执行真实任务：  
点击、输入、操作应用、读取屏幕、整理文件、运行工作流，  
并像数字员工一样完成端到端的复杂任务。

> **我们的使命：让 AI 拥有 *行动能力*，而不仅是对话能力。**

项目正在积极开发中，并以公开方式构建。

---

## 🚀 愿景（Vision）

现代 AI 虽然能回答问题——  
但仍无法稳定、可靠地操作一台真正的电脑。

**IntelliAvatar 的目标，就是改变这一点。**

我们正在构建一个全栈式桌面智能代理，具备：

- 规划多步骤任务  
- 对 Windows 进行视觉级别的 GUI 操作  
- 自主运行工作流  
- 观察 → 判断 → 行动 → 重新规划  
- 管理文件、浏览器、文档与应用  
- 执行计划任务与周期性自动化  
- 随时间建立记忆并持续提升能力  
- 像一名真正的“数字员工”一样工作，而不是聊天界面

虽然仍处于早期，但基础系统正在快速进化。

## 🧠 核心系统支柱（架构概览）

> 当前代码尚未公开，架构稳定后将逐步开放。

- **技能系统（Skills Engine）**  
  文件操作、GUI 操作、浏览器操作、自定义技能

- **规划器（Planner：Intent → Task Graph）**  
  线性 + DAG 任务结构，多阶段推理

- **运行时 / Agent Loop**  
  执行、重试、重新规划、自我纠错

- **感知层（Perception Layer）** *(即将推出)*  
  屏幕内容识别、UI 元素检测、OCR

- **记忆系统（Memory System）**  
  短期 → 长期 → 知识库

- **任务调度器（Scheduler）**  
  周期性任务与自动化流程

- **Shadow Mode（规划中）**  
  实时桌面自主控制

整个系统是模块化设计，目标是扩展成完整的 AI Worker OS。
